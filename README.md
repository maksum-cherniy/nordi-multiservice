# NORDI MULTISERVICE AS - Website

Професійний вебсайт для компанії NORDI MULTISERVICE, яка спеціалізується на лазерному очищенню металу кораблів у Норвегії.

## 🌍 Особливості

- ✅ **Мультимовність** - Норвезька та англійська мови з легким перемиканням
- 📱 **Адаптивний дизайн** - Оптимізовано для всіх пристроїв (мобільні, планшети, комп'ютери)
- 🚀 **Сучасний дизайн** - Професійний та сучасний стиль з градієнтами та анімаціями
- 🗺️ **Інтеграція карти** - Google Maps для показу розташування компанії
- 📞 **WhatsApp інтеграція** - Кнопки для прямого замовлення послуг через WhatsApp
- 💬 **Форма контакту** - Система для отправки повідомлень
- ⭐ **Секція відгуків** - Приклади виконаних робіт з фото "до/після"
- 🎨 **Брендування** - Стильний логотип та кольорова схема

## 🔧 Установка та запуск

### Простий спосіб:

1. Відкрити файл `index.html` безпосередньо в браузері
2. Або скористатися локальним сервером (рекомендується)

### З локальним сервером (Python):

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Потім відкрити в браузері: `http://localhost:8000`

### З Node.js (Live Server):

```bash
npm install -g live-server
live-server
```
